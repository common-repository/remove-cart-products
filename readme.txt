This plugin is useful for removing automatically **cart product** which is added by the users. We have to provide **Five ways** to remove cart products.
Five ways are listed as below :
1. Set Hours or Minutes ( Mandatory )
2. After Login user
3. After Logout User
4. After Close Browser
5. Click on Button

== Description ==
**1. Set Hours or Minutes ( Mandatory )** <br>
As per the time set, the product that is carted will automatically be removed by the plugin if the user will not going to purchase the product ( Hours / Minutes ). As this is a mandatory field, if you do not set these field values then it will take an hour and remove the product automatically that is present in the cart.
**2. After Login user or Open the site** <br>
When users are open the site or login, All cart products will be automatically removed.
**3. After Logout User** <br>
When a user is a logout after that all cart products will be removed automatically.
**4. After Close Browser** <br>
If users are close the whole browser, The product which is added to the cart will be automatically removed.
**5. Click on the button** <br>
When the user clicks on the front end displayed button all cart products will be automatically removed.

== Installation ==

#### From within WordPress

1. Visit 'Plugins > Add New'
2. Search for 'Remove cart products'
3. Activate 'Remove cart products' for WordPress from your 'Plugins' menu from WordPress.

#### Manually

1. Upload the remove-cart-products/ folder to the /wp-content/plugins/ directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= 1. How do I change the hour to minute? =
Go to "clear cart" page and change "Select Time Type" option Hour to Minutes, by default "Hour".

= 2. Can I use button for clear cart product? =
Yes, you can enable button settings in the "clear cart" page and also change button design.

= 3. It is support multi-language? =
Yes, it is support multi-language.
